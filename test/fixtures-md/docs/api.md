# API Reference

Complete API documentation for the project endpoints.

## Endpoints

### GET /health

Returns server health status as JSON.

### POST /data

Submit data for processing. See [the guide](guide.md) for setup details.

## Authentication

Use Bearer tokens for all authenticated endpoints. See <https://example.com/auth> for the spec.
