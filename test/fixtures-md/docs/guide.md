# Getting Started Guide

Welcome to the guide. This document covers basic setup and configuration.

## Prerequisites

You need Node.js 20+ and npm 10+.

## Setup Steps

1. Clone the repository
2. Install dependencies
3. Run the dev server

## Configuration

See [[api-reference]] for detailed config options.

### Environment Variables

Set `PORT` and `HOST` before starting the server.

### Database Setup

Run the migration script first.
